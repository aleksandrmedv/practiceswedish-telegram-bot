# 🇸🇪 Swedish Learning Telegram Bot

An interactive Telegram bot designed to help users learn Swedish in a simple and engaging way.

---

## 🚀 Live Demo

👉 Try the bot: https://t.me/PracticeSwedish_bot

---

## 💡 About the Project

This bot was created as a practical tool for learning Swedish vocabulary, verbs, and grammar.
It is especially useful for beginners and SFI students.

The learning process is built in a **step-by-step interactive format**, making it easy to stay engaged and track progress.

---

## 🔥 Features

* 🧠 Word training (Swedish → English / Russian / Ukrainian)
* 📚 Noun forms training (en/ett, plural, definite forms)
* ⚡ Verb forms training (infinitive, present, preteritum, supinum)
* 🔁 Reinforcement mode (repeat mistakes)
* 📊 Progress tracking (correct answers stats)
* 🌍 Multi-language support

---

## 🎯 Who is it for?

* People learning Swedish from scratch
* SFI students
* Anyone who wants to improve vocabulary and grammar

---

## 🛠 Tech Stack

* Python
* python-telegram-bot
* Telegram Bot API

---

## 📸 How it works

1. Choose your language
2. Select training mode
3. Practice words, verbs, or nouns
4. Track your progress
5. Repeat mistakes to improve
